# Competitive Landscape — analysis spec

**Evaluates:** existing competitors, their positioning and strengths/weaknesses, and whether a defensible gap exists.

**Reads:** PRD 1.3 (offering), 1.5 (competitors the user already knows).

**Method:**
- Map direct **and** indirect competitors — include the "do nothing / status quo" alternative, which is usually the real competitor.
- For the top 3–5: positioning, strengths, weaknesses, pricing posture.
- Name the **defensible gap** the product could own — or honestly conclude there isn't one.

**Writes:** an artifact per `templates/analysis.md`. `verdict` ∈ {clear-gap, crowded, dominated, unclear}. `key_findings`: top competitors · the gap (or its absence) · the strongest incumbent threat.

# Benchmarking Study — analysis spec

**Evaluates:** the proposed model against comparable ventures that won — and ones that died.
**Reads:** PRD 1.3/1.4 + `competitive-landscape`.
**Method:**
- Pick 2–4 real comparables; for each, what worked, what killed it, and the metric to copy.
- **Study the failures hardest** — they reveal the non-obvious traps.
**Writes:** artifact per template. `verdict` ∈ {proven-pattern, mixed, cautionary}. `key_findings`: the pattern to emulate + the failure mode to avoid.

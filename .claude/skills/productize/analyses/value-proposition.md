# Value Proposition Analysis — analysis spec

**Evaluates:** the unique value delivered to customers that competitors don't.
**Reads:** PRD 1.1/1.2/1.3 + `market-study` (segment pull).
**Method:**
- Frame by **jobs-to-be-done**: the progress the customer wants; the pains relieved and gains created.
- Map opportunity → solution: does the offering hit a top pain, or a nice-to-have?
**Writes:** artifact per template. `verdict` ∈ {compelling, moderate, weak}. `key_findings`: the core value + the job it wins on.

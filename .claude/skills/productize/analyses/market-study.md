# Market Study — analysis spec

**Evaluates:** target market size, growth, segments, and demand drivers.

**Reads:** PRD 1.2 (segments), 1.6 (geography), 1.1 (problem → demand).

**Method:**
- Size it both ways — top-down (TAM) and bottom-up (SAM/SOM). Show both; if they diverge wildly, say which you trust and why.
- Identify demand drivers and the trend direction (growing / flat / declining), with reasoning.
- Name the 1–3 segments with the strongest pull.

**Writes:** an artifact per `templates/analysis.md`. `verdict` ∈ {strong-demand, moderate, weak, unclear}. `key_findings`: market-size estimate · strongest segment · biggest demand risk.

# Blue Ocean Strategy — analysis spec

**Evaluates:** uncontested market space where competition is irrelevant.
**Reads:** PRD 1.3 + `competitive-landscape` (the crowded axes).
**Method:**
- Apply Eliminate-Reduce-Raise-Create: which industry givens to drop, which to amplify, which net-new value to add.
- Test the new space for **real demand** — an uncontested market can be empty because no one wants it.
**Writes:** artifact per template. `verdict` ∈ {blue-ocean-found, niche, none}. `key_findings`: the uncontested angle + whether demand is real.

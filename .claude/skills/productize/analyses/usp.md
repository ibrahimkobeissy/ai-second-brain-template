# USP Identification — analysis spec

**Evaluates:** the single most compelling reason a customer chooses this over everything else.
**Reads:** `value-proposition` + `competitive-differentiation`.
**Method:**
- Collapse the value + the defensible edge into **one sentence** a customer would repeat.
- Stress-test it: is it true, specific, and hard for an incumbent to claim?
**Writes:** artifact per template. `verdict` ∈ {sharp, vague, none}. `key_findings`: the USP sentence + whether it survives the stress-test.

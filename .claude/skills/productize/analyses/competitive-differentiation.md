# Competitive Differentiation — analysis spec

**Evaluates:** the features and value where the offering can measurably beat competitors.
**Reads:** PRD 1.3 + `competitive-landscape` (the gap, the incumbents).
**Method:**
- For each axis customers care about, score self vs. top competitors; find where you're genuinely ahead.
- Separate **real** differentiators (hard to copy) from **cosmetic** ones (a sprint away for an incumbent).
**Writes:** artifact per template. `verdict` ∈ {strong-edge, narrow-edge, parity}. `key_findings`: the defensible differentiator(s) + how long the lead lasts.

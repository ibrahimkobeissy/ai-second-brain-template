# SWOT Analysis — analysis spec

**Evaluates:** internal Strengths/Weaknesses against external Opportunities/Threats — a synthesis, not new research.
**Reads:** PRD + `market-study` (opportunities) + `competitive-landscape` (threats).
**Method:**
- Pull each quadrant from evidence already gathered; cite the source finding. Don't invent.
- End with the **2×2 cross-reads** that matter: S×O (where to press), W×T (what to defend).
**Writes:** artifact per template. `verdict` ∈ {favorable, balanced, exposed}. `key_findings`: the one strength to leverage + the one threat to neutralize.

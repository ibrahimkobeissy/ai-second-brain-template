# System · Templates

Reusable **note templates** for Obsidian's Templates / Templater plugins. Point the plugin's "template folder location" setting at this folder.

Seeded: `evergreen` (durable concept note), `literature` (a source in your own words + reaction), `source` (deep analytical pass — Core argument / Key points / Reaction / Connections / Open question).

# Async standup landscape — what we can steal

> [!info] Source
> [11 Best Async Standup Tools (2026)](https://gitmore.io/blog/best-async-standup-tools) — Gitmore

## TL;DR
The market is segmented by *what each tool competes on*: distribution (Geekbot), culture (DailyBot), and automation (Steady, Gitmore). The automation lane — StandupZero's lane — is already contested.

## Why it's relevant to [[standup-tools]]
This is the competitive map for the bet. It tells us the auto-draft positioning is taken and forces the real question: differentiate on *what*, if not automation.

## What we can steal
- **The axes competitors compete on** (distribution / culture / automation) — use them as the columns of our `feature-gap` and `competitive-differentiation` analyses.
- **Pricing anchors** — Geekbot $3, Standuply $2, Gitmore $8.99 — the willingness-to-pay band for `financial-model` and pricing.
- **The buyer = eng manager, value = focus time + visibility** — frame the PRD's value prop around the buyer's outcome, not the bot.
- **The unsolved gap = trust** (a draft people send unedited) — the one place a moat might exist.

## What probably doesn't apply
- DailyBot's mood/kudos/culture play — a different game we're not entering.
- Self-hosted/enterprise (Gitmore) — out of scope for a solo MVP.

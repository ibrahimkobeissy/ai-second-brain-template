# hunter/personal-profile/ (your data — keep private)

Holds your **CV and portfolio**:
- `cv.md` — your structured CV, generated by `job-hunt-cv-from-doc` (the spine the rest of the toolset reads).
- `my-profile.html` — your portfolio page, generated by `job-hunt-craft-profile`.

Drop your CV here as a PDF, run `job-hunt-cv-from-doc`, then `job-hunt-craft-profile`.

> This folder contains real personal data — keep it in a private remote. Never publish your CV.

# 01-work

The **Work** folder contains your active professional Areas of responsibility and ongoing projects.

## Philosophy
- **Strict Separation**: Keep work and personal notes strictly separated to prevent context bleed. 
- **Area Hubs**: Use structured hub notes to organize active efforts.
- **Drafts and Syntheses**: Drop curated drafts here. The `synthesize-drafts` skill will unify them into actionable Strategic Plans.
- **Kanban**: Use `todo-kanban.md` boards within your areas to track actions extracted by the `plan-to-kanban` skill.

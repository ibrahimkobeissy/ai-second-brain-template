# 00-inbox

The **Inbox** is the entry point for all new, unprocessed information. This includes raw captures, web clippings, fleeting notes, and bookmarks. 

## Workflow
1. Drop raw material here.
2. The AI agent (using the `curate-bookmarks` skill) processes items from this folder.
3. The agent extracts the core signal ("what we can steal") and moves the insights into a `draft/` folder within the relevant Area.
4. Once processed, the raw items here are deleted to keep the vault clean and high-signal.
